MY LIBRARY

A simple Javascript app to keep track of the books you've read, or want to read!

--FEATURES--

-Add books through the "Add New Book" button on the left-hand side of your bookshelf.
-Input Title, Author, Book Length, and whether or not you've read the book through the New Book Form.
-Toggle read status by pressing the "Read: Yes" or "Read: No" button located on each book.
-Delete books with the "Delete" button located on each book.

--PURPOSE--

This app was developed to practice object manipulation in the Javascript track of the Odin Project curriculum.
